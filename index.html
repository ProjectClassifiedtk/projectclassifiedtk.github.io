Frequentely Asked Questions

FAQ

Is the game free?

Yes. The game costs no money at all. If you would like to donate, that costs money.

How does donating help?

Donating helps pay for the servers and material used to make the game. Since the game is mostly online, we have to pay for servers and the materials can be models, etc.

What engine is the game made in?

Project Classified is made in the free version of Unity. Unitys programming language is C#.

Who are the people behind the game?

Runabox is the main developer while TheEnthusiastic_16 is the man behind the project.

When will the game release?

The game is set to release some time in 2020.
